# FitLog
Android Studio - FitLog project
Android fitness tracker application built with Android Studio.

Demo video: https://www.youtube.com/watch?v=OW5tDRZ7MFA
## Features
- Add, edit, delete and search workouts
- Statistics – training days, favorite exercise, weekly workouts
- User profile (name, email, weight)
- Step counter using accelerometer sensor
## Tech Stack
- Java
- Android Studio
- Room SQLite Database
- SharedPreferences
- Fragment navigation + BottomNavigationView
## Installation
1. Download the ZIP from the [Releases](https://github.com/tothbalint1221/FitLog/releases) section
2. Extract the ZIP
3. Open in Android Studio (File → Open)
4. Run on emulator or physical device (min. API 24)
